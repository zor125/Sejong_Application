# PRD

## 프로젝트 개요

간호학원 문제집 관리 시스템은 강사가 웹에서 문제집을 제작하고, 특정 기수에 배포하며, 학생이 앱에서 배포된 문제집을 자유롭게 풀이할 수 있도록 지원하는 서비스이다. 강사는 학생의 성적과 풀이 현황을 확인할 수 있다.

## 개발 우선순위

1. 강사용 웹
2. 문제집 제작 및 배포
3. 기수별 학생 관리
4. 성적 확인
5. 학생 앱

## 사용자

### 강사

- 웹에서 문제집을 생성, 수정, 관리한다.
- 문제집을 특정 기수에 배포한다.
- 기수별 학생 목록을 관리한다.
- 학생별 성적과 문제집 풀이 현황을 확인한다.

### 학생

- 앱에서 자신이 속한 기수에 배포된 문제집을 확인한다.
- 배포된 문제집을 자유롭게 푼다.
- 본인의 풀이 결과를 확인한다.

## 핵심 요구사항

- 강사는 웹에서 문제집을 제작할 수 있어야 한다.
- 학생은 앱에서 문제집을 자유롭게 풀 수 있어야 한다.
- 강사는 학생의 성적을 확인할 수 있어야 한다.
- 학생은 기수별로 관리되어야 한다.
- 문제집은 특정 기수에 배포되어야 한다.
- 해당 기수에 속한 학생만 배포된 문제집을 풀 수 있어야 한다.
- 강사용 웹을 우선 개발한다.
- 학생 앱 UI는 Figma Make 코드 스타일을 참고한다.

## 주요 기능 범위

### 강사용 웹

- 로그인
- 대시보드
- 기수 관리
- 학생 관리
- 문제집 목록
- 문제집 제작
- 문제 등록 및 편집
- 문제집 배포
- 성적 및 풀이 현황 확인

### 학생 앱

- 로그인
- 배포된 문제집 목록
- 문제 풀이
- 풀이 결과 확인

## 권한 정책

- 강사는 모든 기수, 학생, 문제집, 성적 정보를 관리한다.
- 학생은 본인이 속한 기수에 배포된 문제집만 조회하고 풀이할 수 있다.
- 학생은 다른 기수의 문제집과 다른 학생의 성적을 조회할 수 없다.

## MVP 범위

- 강사용 웹 로그인
- 기수 생성 및 관리
- 학생 등록 및 기수 배정
- 문제집 생성
- 객관식 문제 등록
- 문제집 기수 배포
- 기수별 성적 조회
